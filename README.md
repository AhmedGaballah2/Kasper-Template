# Kasper Template

Kasper Template is a clean and modern frontend template built with **HTML & CSS only**.  
It is **fully responsive**, working seamlessly on all devices (Desktop, Tablet, Mobile).

---

## 📂 Project Structure

- `index.html` - The main HTML page of the template.  
- `master.css` - Main CSS file for styling the template.  
- Additional assets like images or fonts (if any).

---

## ⚡ Features

- **Responsive Design**: Adapts smoothly to all screen sizes.  
- **HTML & CSS Only**: No JavaScript or external libraries required.  
- **Easy to Customize**: Edit texts, images, and colors easily.

---

## 🛠️ How to Use

1. Copy the project files to a new folder on your computer.  
2. Open `index.html` in any modern browser.  
3. Customize the files as needed (texts, colors, images, etc.).
